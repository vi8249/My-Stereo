# My-Stereo

發想: 

我的老家是一間小小的擴大機工廠，我爸媽在接單時，常常會搞不清楚，家裡的零組件的存貨數量，
於是我在退伍後的這段時間，自學Web Development，用上述構想做出了這個展示專案。

使用 - html, css, javascript, bootstrap, jquery, node.js, express, Mongoose, bcrypt, passport

預計功能: 採購、銷貨、存貨管理。

目前完成: 採購。
